PLATAFORMAS JAVA

JAVA SE (JAVA STANDE EDITION): é a base da plafatorma, cotendo as bibliotecas comuens, sendo direcionado a aplicações de pc's e servidores.
JAVA EE (JAVA ENTERPRISE EDITION): edicição voltada para o desenvolvimento corporativo e aplicações web
JAVA ME (JAVA MICRO EDITION): edição de desenvolvimento de aplicações para dispositivos móveis e embarcados. 
JAVA FX (PLATAFORMA DE SOFTWARE MILTIMÍDIA): é uma tecnologia de software que, ao ser combinada com java, permite a criação e implatação de aplicação de aparencia moderna e conteudo rico de audio e vídeo.
